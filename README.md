# OcrHelper V0.1
一个OCR的软件。可以为答题软件提供快速搜索服务。

## 环境依赖

如果下载双击无法打开，请安装 JRE（JavaRuntime Environment）

+ [下载地址](http://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html)
